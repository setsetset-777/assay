# RC 2

![Status Todo](https://img.shields.io/badge/Status-To%20be%20refined-yellow.svg)

In the second release candidates, an instance of the tool should have an administration tool and allow many contributors to configure many projects for many clients. An instance must have an adminstrator contributor who has access to all instance content.

### An instance has an administration tool
- Am administrator contributor can be created on instance first connexion
- Contributors can be defined as administrator
- Administrators can add/edit/remove contributors

### Forms are related to projects
- Contributors can crete/edit/remove projects
- A project can be associated to:
  - a title
  - a description
  - many forms

### Forms are private
- A contributor can invite a client to edit a form
- Accessing a form needs authentication
- Only a authorized client can edit the form
- A contributor can not edit the form
