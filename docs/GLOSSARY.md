# Glossary

- **the tool**: __the__ tool refer to the current project. It can also be designated as "assay" (the name can change).
- **instance**: a unique occurence of the tool installed on a server.
- **user role**: There are different type of users related to an instance. They can be owner, contributor or client
- **administrator**: a user with administrative role. They own the instance. They can manage all users and projects.
- **contributor**: uses the tool to gather info for their clients.
- **client**: target person of a contributor from which they want to gather input.
- **form**: a type of input acquisation form which consist of a multitude of fields a client fill in with relevant data.
